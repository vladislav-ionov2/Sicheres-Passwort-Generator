Sicheres Passwort Generator:

Kleines Übungsprojekt, das ich selbst gebaut habe, um JavaScript und einfache Web-Techniken zu üben.  
Die Seite erzeugt zufällige Passwörter nach gewählten Einstellungen. Alles ist bewusst einfach gehalten — wie ein Schulprojekt.

---

Was macht das Programm:
- Erzeugt zufällige Passwörter mit gewünschter Länge  
- Optionen: Großbuchstaben, Kleinbuchstaben, Zahlen, Sonderzeichen  
- Passwort wird im Feld angezeigt und kann kopiert werden

---

Funktionen:
- Eingabe der gewünschten Länge (4–30)  
- Checkboxen für verschiedene Zeichentypen  
- Knopf „Passwort erstellen“  
- Knopf „Kopieren“ (kopiert das erzeugte Passwort in die Zwischenablage)  
- Kurze Fehlermeldung, wenn keine Option gewählt ist

---

Wie benutze ich das (einfach):
1. Repository herunterladen oder klonen.  
2. Datei `index.html` im Browser öffnen (kein Server nötig).  
3. Länge einstellen, Optionen wählen.  
4. Auf **Passwort erstellen** klicken.  
5. Bei Bedarf **Kopieren** drücken.

---

Projekt-Dateien:
- `index.html` — Seite / Eingabe / Ausgabe  
- `styles.css` — einfaches Design (Schul-Stil)  
- `app.js` — Logik zum Erzeugen und Kopieren des Passworts

---

Was ich dabei gelernt habe
- Grundlagen von JavaScript (Variablen, Funktionen, Schleifen)  
- Einfache DOM-Manipulation (Elemente lesen und schreiben)  
- Event Listener (Buttons, Eingaben)  
- Einfaches CSS für Layout und Stil

---

Was man später verbessern könnte:
- bessere Zufallsquelle (für Produktions-Nutzung)  
- Anzeige der Passwortstärke  
- Speichern von Favoriten (nicht empfohlen für echte Passwörter)  
- Mehr Mobile-Anpassungen / Accessibility-Verbesserungen

---
Fazit:
Das Projekt ist als Lernübung gedacht. Für echte Anwendungen sollten zusätzliche Sicherheitsmaßnahmen berücksichtigt werden.
